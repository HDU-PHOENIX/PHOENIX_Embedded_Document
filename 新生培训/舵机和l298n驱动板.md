# 舵机和L298N的使用
> 本文介绍如何使用舵机，L298N驱动板控制TT马达
## 1. 舵机的使用
### 1.1 舵机简介
舵机是一种能够精确控制角度位置的电机，简单来说就是通过输入PWM信号来控制角度的一个电机
### 1.2 舵机接线
舵机一般有三根线，分别是电源线、地线和信号线。通常情况下：
-   **红色**：连接到电源正极
-   **黑色**：连接到电源负极
-   **黄色**：连接到MCU的PWM引脚，用于接收控制信号
### 1.3 舵机定时器设置
-   **频率**：50Hz（周期20ms）
-   **占空比**：范围是0.5ms-2.5ms，180°舵机按如下表格映射
| 占空比 | 角度 |
| :---: | :---: |
| 0.5ms | 0° |
| 1.0ms | 45° |
| 1.5ms | 90° |
| 2.0ms | 135° |
| 2.5ms | 180° |
> 注意这个表格是针对180°舵机的，不同型号的舵机占空比和角度的对应关系可能会有所不同，具体请参考舵机的规格说明书。
### 1.4 舵机控制代码示例
```C
    gimbal_yaw_compare = gimbal_yaw_compare + yaw * GIMBAL_YAW_COEFFICIENT;
    if(gimbal_yaw_compare < GIMBAL_YAW_MIN_COMPARE)
        gimbal_yaw_compare = GIMBAL_YAW_MIN_COMPARE;
    else if(gimbal_yaw_compare > GIMBAL_YAW_MAX_COMPARE)
        gimbal_yaw_compare = GIMBAL_YAW_MAX_COMPARE;
    __HAL_TIM_SET_COMPARE(&htim2, TIM_CHANNEL_1, (int16_t)gimbal_yaw_compare);
```


## 2. L298N驱动板使用
### 2.1 tt马达简介
TT马达是一种常用的小型直流电机。TT马达通常配备有减速箱，然后一般使用PWM信号来控制其转速和方向。直流电机不关心脉宽时间，只关心$平均电压大小 \approx 占空比 × 电源电压$
### 2.2 为什么要使用L298N驱动板
-   **电流不够**:GPIO仅能输出逻辑电平，无法提供电流和足够功率驱动马达
-   **电压不匹配**:MCU的GPIO 3.3V/5V，但电机可能需要6V/12V
-   **电机反电动势**:电机反向电动势会冲击 MCU，导致死机甚至烧坏
### 2.3 L298N驱动板引脚说明及功能
![实物图](https://youke1.picui.cn/s1/2025/11/17/691aeaf7e688a.png)
注意一个L298N驱动板可以控制两个直流电机，分别称为A通道和B通道。
#### 2.3.1 ENA/ENB
也就是图上被跳线帽接到`+5v`的两个引脚，用于使能A、B通道的马达驱动。
我们是往这两个引脚输入**PWM信号**来控制马达的转速，ENA控制A通道马达，ENB控制B通道马达。
> 所以在使用的时候需要把这个跳线帽拔了，然后把ENA/ENB接到MCU的PWM引脚上。
#### 2.3.2 IN1/IN2/IN3/IN4
IN1和IN2用于控制A通道马达的转动方向，IN3和IN4用于控制B通道马达的转动方向。由于接线的不同，正反向没有一个固定的规律，需要通过实验来确定。
| IN1 | IN2 | 控制方向 |
| :---: | :---: | :---: |
| 低电平 | 低电平 | 停止 |
| 低电平 | 高电平 | 正转 |
| 高电平 | 低电平 | 反转 |
| 高电平 | 高电平 | 停止(不要使用) |

由于IN3/IN4和IN1/IN2的控制逻辑是一样的，只是对应输出的端口不同，这里就不再赘述。
#### 2.3.3 电源
L298N驱动板的电源输入端口是那个三端接线座，分别是`Vin`、`GND`和`+5V OUT`，其中最左侧(靠近板边缘)的是`Vin`，中间是`GND`，最右侧(靠近排针)的是`+5V OUT`。
接线的话，`Vin`接电机电源正极，`GND`接电机电源负极，`+5V OUT`是输出端，可以给MCU提供5V供电，但是不推荐使用，因为输出电流不稳定，有烧mcu的风险。
#### 2.3.4 输出端
最后剩下的两个接线座子连接tt马达即可，由于tt马达的正负只影响转动方向，所以不用特意区分正负极。
### 2.4 L298N驱动板接线示例
![](https://youke1.picui.cn/s1/2025/11/17/691af44837fdd.png)
大概这样，注意绿线表示PWM信号线连接ENA/ENB，蓝紫线连接IN1/IN2控制方向，红黑线连接电源
### 2.5 L298N驱动板示例代码
```C
    // 控制电机(设置占空比以及正方向)
    // 设置左前电机
    if(motor_lf > 0)
    {
        HAL_GPIO_WritePin(GPIOB, GPIO_PIN_0, GPIO_PIN_SET);
        HAL_GPIO_WritePin(GPIOB, GPIO_PIN_1, GPIO_PIN_RESET);
        __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, motor_lf);
    }
    else
    {
        HAL_GPIO_WritePin(GPIOB, GPIO_PIN_0, GPIO_PIN_RESET);
        HAL_GPIO_WritePin(GPIOB, GPIO_PIN_1, GPIO_PIN_SET);
        __HAL_TIM_SET_COMPARE(&htim1, TIM_CHANNEL_1, -motor_lf);
    }
```
